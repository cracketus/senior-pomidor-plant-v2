# Support

This is an alpha edge-node project. Support is best-effort.

Use GitHub issues for:

- Reproducible bugs.
- Documentation gaps.
- Setup problems with mock mode or Raspberry Pi hardware mode.
- Schema or contract questions.

Before opening an issue, include:

- Operating system and hardware model.
- Whether `MOCK_SENSORS` is `true` or `false`.
- Relevant command output with secrets removed.
- Sanitized `.env` values for the affected settings.

Do not post credentials, private network names, raw private telemetry, or plant photos that should not be public.
