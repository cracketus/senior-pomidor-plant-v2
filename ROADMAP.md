# Roadmap

## v0.1.0-alpha

- Publish the Raspberry Pi edge-node foundation.
- Keep telemetry, event, and photo contracts stable enough for early Core integration.
- Document hardware setup, calibration, local buffering, maintenance, and network-health behavior.

## Next

- Add generated API and schema reference documentation.
- Publish privacy-safe dashboard screenshots and hardware photos.
- Expand example payloads and public status-page documentation.
- Add a public dataset roadmap with privacy and redaction policy.
- Add optional lockfile support if exact cross-platform environments become necessary.

## Out of Scope for This Repository

- Core server implementation.
- Database migrations and dashboard services.
- AI/VLM processing.
- Runtime state estimation.
- Actuation and autonomous control.
